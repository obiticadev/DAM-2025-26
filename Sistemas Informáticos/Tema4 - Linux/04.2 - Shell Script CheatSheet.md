# 🐚 Cheatsheet: Introducción a Shellscript (Completo + Funciones)

## 1. Operaciones Aritméticas: `$(( ))` vs `(( ))`

| Sintaxis | Uso | Ejemplo |
| :--- | :--- | :--- |
| **`$(( expresión ))`** | **Obtener valor.** Devuelve el resultado para asignar o mostrar. | `res=$(( a + b ))` |
| **`(( expresión ))`** | **Acción/Lógica.** Modifica variables o evalúa en `if/while`. | `(( contador++ ))` |

## 2. Estructuras de Control: Condicionales

### A. Bloque `if`
```bash
if [ $num -gt 10 ]; then
    echo "Mayor (usando test)"
elif (( num == 10 )); then
    echo "Es 10 (usando aritmética)"
else
    echo "Menor"
fi
```

### B. Bloque `case` (Selector)
```bash
case $variable in
    "opcion1") comandos ;;
    "opcion2"|"opcion3") comandos ;;
    *) comandos_por_defecto ;;
esac
```

## 3. Estructuras de Control: Bucles

### A. Bucle `while` (Mientras sea cierto)
```bash
while [ condición ]; do
    comandos
done
```

### B. Bucle `for`
*   **Lista:** `for var in 1 2 3; do ... done`
*   **Rango:** `for i in {1..10}; do ... done`
*   **Estilo C:** `for (( i=0; i<10; i++ )); do ... done`

## 4. Tabla de Operandos (Comparadores)

| Significado | Numérico `[ ]` | Numérico `(( ))` | Cadenas `[ ]` |
| :--- | :--- | :--- | :--- |
| **Igual** | `-eq` | `==` | `==` |
| **Distinto** | `-ne` | `!=` | `!=` |
| **Mayor que** | `-gt` | `>` | *n/a* |
| **Menor que** | `-lt` | `<` | *n/a* |
| **Mayor o igual** | `-ge` | `>=` | *n/a* |
| **Menor o igual** | `-le` | `<=` | *n/a* |

## 5. Funciones en Bash

Permiten encapsular código para reutilizarlo.

### Definición y Llamada
```bash
# Definición
nombre_funcion() {
    echo "Hola desde la función"
}

# Llamada (sin paréntesis)
nombre_funcion
```

### Paso de Parámetros
Las funciones reciben argumentos como si fueran scripts independientes.
```bash
saludar() {
    echo "Hola $1, tienes $2 años"
}

saludar "Juan" 25
```
*   `$1, $2...`: Argumentos por posición.
*   `$@`: Todos los argumentos.
*   `$#`: Número de argumentos pasados.

### Variables Locales
Evitan modificar variables globales del script.
```bash
mi_funcion() {
    local variable_interna="Solo existo aquí"
}
```

### Retorno de Valores
*   **`return N`**: Devuelve un código de estado (0-255). No sirve para devolver resultados numéricos grandes.
*   **`echo`**: Se usa para devolver datos capturándolos con sustitución de comandos.

```bash
sumar() {
    echo $(( $1 + $2 ))
}

resultado=$(sumar 10 5)
```

## 6. Variables de Entorno y Sistema comunes
*   `$0`: Nombre del script.
*   `$?`: Código de salida del último comando (0 = éxito).
*   `$$`: PID del proceso actual.
*   `$USER`: Usuario actual.
*   `$PWD`: Directorio actual.
