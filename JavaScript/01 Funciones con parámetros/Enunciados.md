# Funciones con Parámetros

### 1. Saludo Simple
Crea una función llamada **`sayHello`** que reciba un **nombre** por parámetro y diga por consola: 
*   `"hola [nombre]"`

### 2. Cálculo de Áreas
*   **Área de un Cuadrado:** Crea la función **`calculateSquareArea`** que calcule el área pasándole la medida del **lado**.
*   **Área de un Triángulo:** Crea la función **`calculateTriangleArea`** que calcule el área pasándole la **base** y la **altura**.
*   **Área de un Círculo:** Crea la función **`calculateCircleArea`** que calcule el área pasándole el **radio**.

### 3. Conversión de Temperatura
*   **Celsius a Fahrenheit:** Crea la función **`celsiusToFahrenheit`** que reciba grados Celsius y los convierta.
    *   *Fórmula:* `ºF = ºC x 1.8 + 32`
*   **Fahrenheit a Celsius:** Crea la función inversa **`fahrenheitToCelsius`**.
    *   *Fórmula:* `ºC = (ºF - 32) / 1.8`

### 4. Cálculo de Impuestos
Crea una función llamada **`totalPrice`** que reciba un **precio** y le sume el IVA (puedes asumir un 21% si no se especifica).

### 5. Generador de Mensajes de Pedido
Crea una función llamada **`writeMessage`** que reciba cuatro parámetros: **`name`**, **`material`**, **`size`** y **`note`**.
*   La función debe escribir por consola el siguiente mensaje:
    *   `"[name] ha pedido una caja de [material] de tamaño [size]. [note]"`
