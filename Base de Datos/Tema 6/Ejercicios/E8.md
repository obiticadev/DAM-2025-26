# Estructura de Cursores

> **NOTA:** Los tipos, variables y cursores deben declararse previamente en el bloque `DECLARE`.

## Formas de manejo de cursores

### FORMA 1: Manejo Explícito
```sql
open nombre_cursor;
fetch nombre_cursor into variables;
while nombre_cursor%found loop
    -- realizar operaciones
    fetch nombre_cursor into variables;
end loop;
close nombre_cursor;
```

### FORMA 2: Bucle FOR de Cursor (Implícito)
```sql
for i in nombre_cursor loop
    -- realizar operaciones con i.nombre_columna
end loop;
```
