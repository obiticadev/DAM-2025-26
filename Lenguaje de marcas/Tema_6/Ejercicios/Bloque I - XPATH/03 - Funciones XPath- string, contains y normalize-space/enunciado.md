# Ejercicio 03: Funciones de cadena
Escribe las expresiones XPath necesarias para `empleados.xml` utilizando funciones de biblioteca:

- **a)** Seleccionar empleados cuyo email contenga "empresa.com".
- **b)** Seleccionar empleados cuyo puesto empiece por "Desar".
- **c)** Obtener la longitud de la cadena del nombre del primer empleado (tal cual aparece en el XML).
- **d)** Obtener el nombre del primer empleado eliminando espacios en blanco extra.
- **e)** Concatenar el nombre y el puesto del segundo empleado con un guion separador.
