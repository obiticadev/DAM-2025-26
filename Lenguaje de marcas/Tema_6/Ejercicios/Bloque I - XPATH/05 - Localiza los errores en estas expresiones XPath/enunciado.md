# Ejercicio 05: Depuración de errores
Identifica el error sintáctico o semántico en cada expresión y escribe la versión corregida:

- **a)** `catalogo/libro` (Cuando se pretende seleccionar desde la raíz).
- **b)** `//libro[@id=001]` (Buscando un ID de tipo cadena).
- **c)** `//libro[autor="Ruiz"]` (Considerando el contexto de uso dentro de un atributo XSLT).
- **d)** `//libro/[precio>15]`
- **e)** `count//libro`
- **f)** `//libro[1,2]` (Para seleccionar las dos primeras posiciones).
