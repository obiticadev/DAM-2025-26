# Ejercicio 17: HTML con clases dinámicas
Crea una tabla HTML a partir de `temperaturas.xml` aplicando estilos CSS según el valor.

**Lógica de clases CSS para las filas (<tr>):**
- `frio`: Si la temperatura es < 10.
- `templado`: Si la temperatura está entre 10 y 25.
- `calor`: Si la temperatura es > 25.

**Nota:** Usa `<xsl:attribute name="class">` dentro de la etiqueta `<tr>`.
