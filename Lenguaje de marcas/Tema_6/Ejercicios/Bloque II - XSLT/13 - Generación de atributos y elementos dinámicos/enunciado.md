# Ejercicio 13: Atributos dinámicos y AVT
Crea una plantilla para los libros del catálogo que genere un enlace HTML (`<a>`).

**Requisitos:**
- El atributo `href` debe tener este formato: `libro-[ID].html` (donde [ID] es el atributo id del libro).
- El texto del enlace debe ser el título del libro.
- **Reto:** Intenta hacerlo de dos formas: usando `<xsl:attribute>` y usando llaves `{}` (Attribute Value Templates - AVT).
