# Ejercicio 10: Lógica condicional
Toma como base la estructura del ejercicio anterior y añade la siguiente lógica:

- **Salarios:** Si el salario es superior a 3000€, debe mostrarse en **negrita** y añadir el texto "(Senior)".
- **Departamento:** Si un empleado no tiene departamento asignado (o el campo está vacío), debe mostrar el texto "Sin asignar".
- Utiliza `<xsl:if>` o `<xsl:choose>` según convenga.
