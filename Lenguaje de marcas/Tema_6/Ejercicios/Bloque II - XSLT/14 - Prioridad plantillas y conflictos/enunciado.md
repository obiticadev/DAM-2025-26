# Ejercicio 14: Prioridades de plantillas
Analiza el archivo `prioridades.xsl`. Hay varias plantillas que coinciden con el mismo nodo `<libro>`.

**Preguntas a resolver:**
- ¿Qué prioridad tiene por defecto cada una?
- ¿Qué ocurre si un libro es hijo de `catalogo` y además tiene `disponible="true"`? (Conflicto).
- ¿Cómo usarías el atributo `priority` para asegurar que una plantilla gane sobre otra?
