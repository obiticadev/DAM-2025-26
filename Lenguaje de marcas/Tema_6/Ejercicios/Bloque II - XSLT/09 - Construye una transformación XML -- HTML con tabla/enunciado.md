# Ejercicio 09: Transformación XML a Tabla HTML
Escribe una hoja XSLT completa que transforme el archivo `empleados.xml` en una tabla HTML.

**Requisitos:**
- Los empleados deben aparecer ordenados alfabéticamente por **nombre**.
- La tabla debe tener tres columnas: **Nombre**, **Departamento** y **Salario**.
- Define correctamente la salida como `method="html"`.
